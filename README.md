
# 🎬 Netflix Clone (HTML & CSS Only)

A simple **Netflix Clone** built using only **HTML5** and **CSS3** for learning purposes. This project recreates the basic Netflix homepage layout and is intended to practice frontend design and styling.

## 🚀 Features

- Netflix-inspired homepage UI
- Static layout
- Clean and simple design
- Built using only HTML and CSS

## 🛠️ Tech Stack

- HTML5
- CSS3

## 📂 Project Structure

```text
Netflix-Clone/
│── index.html
│── style.css
│── README.md
```

## ▶️ How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/netflix-clone.git
   ```

2. Open the project folder.

3. Open `index.html` in your preferred web browser.

## 📌 Note

- HTML & CSS only
- No JavaScript
- No backend
- No database
- No API integration
- Not responsive
- Created for learning and practice only
